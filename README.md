# RedisAdmin
A Redis Web Manager for debug && statistics  real time based on Tornado

###RedisAdmin有哪些功能？

* 时时查看 `Redis`运行信息
  *  Top keys信息
  *  Top comand信息
  *  Memory consumption 信息
  *  ......
* 查看单独KEY的VALUE信息，方便调试
* 统计目前KEY占用的内存，方便容量预估
* 更多功能...
